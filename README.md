**VersionIt** is a script to change the version in several places:

* Xcode project version and build number
* Settings.bundle first DefaultValue
* CHANGELLOG.md [Unreleased] heading
* podspec version
* git tag

## Installation

    $ chmod 755 versionit
    $ mv versionit /usr/local/bin

## Usage

See help:

    $ versionit -h
